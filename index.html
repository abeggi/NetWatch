Features
01 // SCAN

Intelligent scanning

Detects hosts on your subnet with port 22 open via Nmap. Resolves hostnames through reverse DNS with SSH fallback.

02 // TERMINAL

Browser SSH terminal

Click any host card to open a full interactive shell in a new tab. Built on WebSockets + Paramiko. No extra client needed.

03 // CACHE

Persistent cache

Last scan results are displayed immediately on load. The grid is always populated, even before a new scan completes.

04 // CONNECT

Quick-Connect bar

Manually enter any IP to open a terminal without scanning first. Useful for known hosts outside the current subnet.

05 // CONFIG

Settings modal

Configure subnet and default SSH credentials directly from the UI. Persisted to settings.json on disk.

06 // DAEMON

systemd service

Run as a system daemon with auto-restart on failure and autostart on boot. Managed via the bundled manage.sh script.

Installation

Up in three commands.

Clone the repo, install Python dependencies, and optionally configure your subnet and SSH credentials.

Nmap must be present on the host system. The manage.sh script handles the full systemd lifecycle: install, enable, start, stop, status.

$ git clone https://github.com/abeggi/NetWatch $ cd terminal && pip install -r requirements.txt $ cp settings.example.json settings.json   # run as systemd service $ sudo ./manage.sh install $ sudo ./manage.sh enable $ sudo ./manage.sh service-start   # open http://localhost:8000
